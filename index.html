IOT Humidity Monitoring

Aug 2025 - Nov 2025

This project was completed for the Internet of Things subject where I was apart of a 2 man team in which we created an ESP32 hosted application which could monitor and control the humidity of a room. The application continuously collected environmental data such as humidity and temperature using onboard sensors and used this data to make intelligent control decisions in real time.

Throughout the project, I architected the ESP32 hosted webserver. This webserver included sensor integration and the control logic used to automatically activate or deactivate the dehumidifier based on predefined thresholds. Features such as live humidity monitoring, automatic control, and data logging were implemented to address practical user needs around efficiency and indoor comfort. Graphs and charts where displayed to the user outlining the rooms current temperature, humidity and pressure while allowing them to set thresholds of when to turn on and off the dehumidifier based on these metrics.

Overall, this project strengthened my skills in IoT system design and embedded development while reinforcing my ability to translate real-world problems into reliable technical solutions. It demonstrates my capability to design, build, and deploy a complete hardware and software system, combining low-level device control with higher level system thinking.
